# Create Vetradocs Backend

The official scaffolding tool for creating backends for **Vetradocs** (AI-powered documentation).

This tool creates a backend service that connects your documentation site (VitePress, Docusaurus, or Scalar) to an AI model.

## Usage

```bash
npx create-vetradocs-backend@latest
```

## Features

- **Interactive**: Easy-to-use CLI prompts.
- **Templates**:
  - **Cloudflare Workers**: Serverless, low-latency AI using Cloudflare Workers AI.
  - **Node.js (Express)**: Standard server supporting OpenAI, Ollama, Anthropic, etc.
- **Ready-to-deploy**: pre-configured with necessary scripts and configuration files.
